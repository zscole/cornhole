On-chain skill-based tossing game where players compete for $CORN prizes.

## Concept
Cornhole brings the beloved backyard game on-chain with physics-based gameplay, competitive rankings, and real stakes. Land bags on the board (1 pt), sink the hole (3 pts), and climb to Legend status.

## Key Features
- **1v1 Matches**: Ranked ELO system, high-roller stakes
- **Tournaments**: Daily, weekly, and seasonal championships
- **$CORN Token**: Entry fees, prizes, staking rewards
- **NFT Cosmetics**: Custom bags, boards, trails, avatars
- **Anti-Cheat**: Commit-reveal mechanics, on-chain verification
- **Cross-platform**: Web + mobile support
